Box-sizing: border-box:
Se recomienda ponerlo para todo. 

Tenemos que tapar los estilos predeterminados.
Para no tener problemas  a futuro, tenemos que utilizar ciertas etiquetas en todo, para eso utilizamos =
* {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}
con esto nos permite quitar los estilos predeterminados y no tener problemas a futuro con ciertas etiquetas.
* es el selector universal (es decir todo.)
